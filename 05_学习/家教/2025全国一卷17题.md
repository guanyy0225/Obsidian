### 解答

#### (1) 证明：平面 $PAB \perp$ 平面 $PAD$

**证明：**
根据题设条件，$PA \perp \text{平面 } ABCD$。
因为直线 $AD$ 在平面 $ABCD$ 内，所以 $PA \perp AD$。
又因为题设条件 $AB \perp AD$。
因为 $PA \subset \text{平面 } PAB$，$AB \subset \text{平面 } PAB$，且 $PA \cap AB = A$。
根据==直线与平面垂直的判定定理==，由于 $AD$ 垂直于平面 $PAB$ 内的两条相交直线 $PA$ 和 $AB$，可得 $AD \perp \text{平面 } PAB$。
又因为直线 $AD \subset \text{平面 } PAD$。
根据==平面与平面垂直的判定定理==，一个平面经过另一个平面的一条垂线，则这两个平面互相垂直。
因此，**平面 $PAB \perp$ 平面 $PAD$**。

### 一、直线与平面垂直的定义

*   **定义**：如果一条直线和一个平面内的**任意一条**直线都垂直，那么我们就说这条直线和这个平面互相垂直。
    *   **符号语言**：若对于平面 $\alpha$ 内的任意直线 $a$，都有 $l \perp a$，则 $l \perp \alpha$。
    *   **说明**：定义本身无法直接用于判定，因为你不可能去验证平面内无穷无尽的所有直线。判定定理的作用就是提供一个“捷径”，让我们只需验证有限的几个条件就能得出结论。

---

### 二、直线与平面垂直的判定定理

这是在证明题目时最常用的工具。

#### 判定定理 1 (最基本、最常用)

*   **定理内容**：如果一条直线与一个平面内的**两条相交直线**都垂直，那么该直线与此平面垂直。
*   **符号语言**：
    若 $l \perp a$，$l \perp b$，且 $a \subset \alpha$，$b \subset \alpha$，$a \cap b = P$，
    则 $l \perp \alpha$。
*   **核心要点（缺一不可）**：
    1.  **线线垂直**：直线 $l$ 必须垂直于平面内的**两条**直线 $a$ 和 $b$。
    2.  **线在面内**：这两条直线 $a$ 和 $b$ 都必须在目标平面 $\alpha$ 内。
    3.  **两线相交**：这两条直线 $a$ 和 $b$ **必须相交**。如果它们平行，则无法判定。
*   **适用场景**：这是证明线面垂直最直接、最经典的方法，尤其适用于可以在平面内轻松找到或作出两条与已知直线垂直的相交线的图形（如长方体、正方体、棱锥等）。

#### 判定定理 2 (涉及面面垂直)

*   **定理内容**：如果两个平面互相垂直，那么在一个平面内**垂直于它们交线**的直线，垂直于另一个平面。
*   **符号语言**：
    若 $\alpha \perp \beta$，$\alpha \cap \beta = a$，且 $l \subset \alpha$，$l \perp a$，
    则 $l \perp \beta$。
*   **核心要点**：
    1.  **面面垂直**：必须以两个平面互相垂直为**前提条件**。
    2.  **线在面内**：直线 $l$ 必须在其中一个平面 $\alpha$ 内。
    3.  **垂直交线**：直线 $l$ 必须垂直于两个平面的**交线** $a$。
*   **适用场景**：当题目条件中已经给出了面面垂直，或者可以很方便地证明出面面垂直时（例如，涉及直二面角、图形翻折等问题），使用此定理非常高效。

### 三、直线与平面垂直的判定方法 (向量法)

在解析几何（坐标系法）中，判定方法变得非常直观。

*   **判定方法**：如果一条直线的**方向向量**与一个平面的**法向量平行（共线）**，那么该直线垂直于此平面。
*   **符号语言**：
    设直线 $l$ 的方向向量为 $\vec{v}$，平面 $\alpha$ 的法向量为 $\vec{n}$。
    若 $\vec{v} \parallel \vec{n}$ (即存在非零实数 $k$ 使得 $\vec{v} = k\vec{n}$)，
    则 $l \perp \alpha$。
*   **核心要点**：
    1.  建立空间直角坐标系。
    2.  求出直线的方向向量。
    3.  求出平面的法向量。
    4.  判断两个向量是否共线。
*   **适用场景**：在能够方便建立坐标系的图形中，这是一种“万能”且强大的方法，能将复杂的空间想象问题转化为代数计算，思路清晰，不容易出错。

---

此外，还有一些由线面垂直引申出的**性质定理**，虽然不是判定定理，但在解题中也经常使用：

*   **性质1**：如果一条直线垂直于一个平面，那么它垂直于这个平面内的**所有**直线。（这是定义的直接应用）
*   **性质2**：如果两条直线都垂直于同一个平面，那么这两条直线互相平行。
*   **性质3**：如果两个平面都垂直于同一条直线，那么这两个平面互相平行。

#### (2) 

由题意 $PA \perp \text{平面 } ABCD$ 且 $AB \perp AD$，可知直线 $PA, AB, AD$ 两两垂直。
故可以以点 $A$ 为原点，分别以 $\overrightarrow{AB}, \overrightarrow{AD}, \overrightarrow{AP}$ 的方向为 $x, y, z$ 轴的正方向，建立如下图所示的空间直角坐标系 $A-xyz$。

根据题设长度，$PA = AB = \sqrt{2}$，$AD = \sqrt{3}+1$，$BC = 2$。
各点坐标如下：
$A(0, 0, 0)$
$B(\sqrt{2}, 0, 0)$
$P(0, 0, \sqrt{2})$
$D(0, \sqrt{3}+1, 0)$

为了确定点 $C$ 的坐标，我们分析底面 $ABCD$。因为 $BC \parallel AD$ 且 $AB \perp AD$，所以 $ABCD$ 是一个直角梯形，且 $AB \perp BC$。
设点 $C$ 的坐标为 $(x_c, y_c, z_c)$。
因为 $C$ 在平面 $ABCD$ 上，所以 $z_c = 0$。
向量 $\overrightarrow{BC} = (x_c - \sqrt{2}, y_c, 0)$。
因为 $AB \perp BC$，所以 $\overrightarrow{AB} \cdot \overrightarrow{BC} = 0$。
$(\sqrt{2}, 0, 0) \cdot (x_c - \sqrt{2}, y_c, 0) = \sqrt{2}(x_c - \sqrt{2}) = 0 \implies x_c = \sqrt{2}$。
又因为 $|BC| = 2$，所以 $|\overrightarrow{BC}| = \sqrt{(\sqrt{2}-\sqrt{2})^2 + y_c^2 + 0^2} = \sqrt{y_c^2} = |y_c| = 2$。
由图可知 $y_c > 0$，故 $y_c = 2$。
所以点 $C$ 的坐标为 $(\sqrt{2}, 2, 0)$。

#### (i) 证明：$O$ 在平面 $ABCD$ 上

设球心 $O$ 的坐标为 $(x, y, z)$，球半径为 $R$。
因为点 $P, B, C, D$ 均在球面上，所以球心 $O$ 到这四点的距离相等。
$|OP|^2 = |OB|^2 = |OC|^2 = |OD|^2 = R^2$

由 $|OP|^2 = |OB|^2$ 可得：
$$x^2 + y^2 + (z - \sqrt{2})^2 = (x - \sqrt{2})^2 + y^2 + z^2$$
$$x^2 + y^2 + z^2 - 2\sqrt{2}z + 2 = x^2 - 2\sqrt{2}x + 2 + y^2 + z^2$$
$$-2\sqrt{2}z = -2\sqrt{2}x \implies z = x$$

由 $|OB|^2 = |OC|^2$ 可得：
$$(x - \sqrt{2})^2 + y^2 + z^2 = (x - \sqrt{2})^2 + (y - 2)^2 + z^2$$
$$y^2 = y^2 - 4y + 4 \implies 4y = 4 \implies y = 1$$

由 $|OD|^2 = |OC|^2$ 可得：
$$x^2 + (y - (\sqrt{3}+1))^2 + z^2 = (x-\sqrt{2})^2 + (y-2)^2 + z^2$$
将 $y=1$ 代入：
$$x^2 + (1 - \sqrt{3} - 1)^2 = (x-\sqrt{2})^2 + (1-2)^2$$
$$x^2 + (-\sqrt{3})^2 = x^2 - 2\sqrt{2}x + 2 + (-1)^2$$
$$3 = -2\sqrt{2}x + 3 \implies -2\sqrt{2}x = 0 \implies x = 0$$

因为 $z = x$，所以 $z = 0$。
由于球心 $O$ 的 $z$ 坐标为 $0$，所以点 $O$ 位于 $xy$ 平面。
在我们建立的坐标系中，$xy$ 平面即为平面 $ABCD$。
因此，**球心 $O$ 在平面 $ABCD$ 上**。
此时，球心 $O$ 的坐标为 $(0, 1, 0)$。

#### (ii) 求直线 $AC$ 与直线 $PO$ 所成角的余弦值

设直线 $AC$ 与直线 $PO$ 所成的角为 $\theta$。我们可以利用向量的点积公式来求解。
$\cos\theta = \frac{|\overrightarrow{AC} \cdot \overrightarrow{PO}|}{|\overrightarrow{AC}| \cdot |\overrightarrow{PO}|}$

1.  **求方向向量：**
    $\overrightarrow{AC} = C - A = (\sqrt{2}, 2, 0)$
    $\overrightarrow{PO} = O - P = (0, 1, 0) - (0, 0, \sqrt{2}) = (0, 1, -\sqrt{2})$

2.  **计算向量点积：**
    $\overrightarrow{AC} \cdot \overrightarrow{PO} = (\sqrt{2})(0) + (2)(1) + (0)(-\sqrt{2}) = 2$

3.  **计算向量模长：**
    $|\overrightarrow{AC}| = \sqrt{(\sqrt{2})^2 + 2^2 + 0^2} = \sqrt{2 + 4} = \sqrt{6}$
    $|\overrightarrow{PO}| = \sqrt{0^2 + 1^2 + (-\sqrt{2})^2} = \sqrt{0 + 1 + 2} = \sqrt{3}$

4.  **计算余弦值：**
    $$\cos\theta = \frac{|2|}{\sqrt{6} \cdot \sqrt{3}} = \frac{2}{\sqrt{18}} = \frac{2}{3\sqrt{2}}$$
    化简分母：
    $$\cos\theta = \frac{2\sqrt{2}}{3\sqrt{2}\sqrt{2}} = \frac{2\sqrt{2}}{3 \cdot 2} = \frac{\sqrt{2}}{3}$$

**答：** 直线 $AC$ 与直线 $PO$ 所成角的余弦值为 $\frac{\sqrt{2}}{3}$。
